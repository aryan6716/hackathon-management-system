# Hackathon Management System

A secure and scalable web-based Hackathon Management System that enables organizers to create hackathons, participants to form teams and submit projects, and judges to evaluate submissions.

---

## 🚀 Features

- User registration and login
- Role-based access (Admin, Participant, Judge)
- Hackathon creation and management
- Team creation and joining
- Project submission system
- Judge evaluation system
- Leaderboard and results

---

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/aryan6716/hackathon-management-system.git
```

Go to project folder

```
cd hackathon-management-system
```

Install dependencies

```
npm install
```

Run server

```
npm run dev
```
